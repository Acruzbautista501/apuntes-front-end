# 📚 Mis Guías de Desarrollo Frontend

Bienvenido a mi repositorio personal de conocimientos. Aquí documento conceptos clave, snippets de código y mejores prácticas sobre el stack moderno que utilizo en mis proyectos.

## 🛠️ Lenguajes y Lógica
### [TypeScript Esencial]
<!-- (./typescript-esencial) -->
Dominio de tipos, interfaces, genéricos y decoradores. Todo enfocado a crear aplicaciones robustas y evitar errores en tiempo de ejecución.


## 🎨 Maquetación y Estilos
### [CSS3 Avanzado]
<!-- (../css/layout) -->
* **Flexbox & Grid:** Control total del layout responsivo.
* **SASS:** Uso de variables, mixins y nesting para CSS escalable.
* **Bootstrap 5:** Implementación rápida de componentes y utilidades.

### [Tailwind CSS 4]
<!-- (../css/tailwind) -->
Exploración de la nueva versión "CSS-first", configuración de temas mediante variables de CSS y optimización de bundles.


## ⚡ Frameworks y Ecosistema
### [Vue.js 3 (Composition API)]
<!-- (../frameworks/vue-3) -->
Uso profundo de `setup`, `ref`, `reactive` y el sistema de reactividad con **TypeScript**. Patrones de diseño para componentes reutilizables.

### [Nuxt]
<!-- (../frameworks/nuxt) -->
Renderizado del lado del servidor (SSR), generación de sitios estáticos (SSG) y manejo automático de rutas y módulos.

### [Angular]
<!-- (../frameworks/angular) -->
Estructura de módulos, servicios, inyección de dependencias y el manejo de RxJS para flujos de datos asíncronos.
