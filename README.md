# Apuntes de Frontend Development

Bienvenido a mi repositorio personal de documentación técnica y hojas de referencia (*cheat sheets*). Aquí centralizo conceptos clave, mejores prácticas y ejemplos de implementación sobre las tecnologías que utilizo para construir interfaces web modernas, escalables y tipadas.

## Contenido del Repositorio

El repositorio está organizado por módulos para facilitar la consulta rápida de sintaxis y lógica de programación:

### 1. **TypeScript 📘**
Guías sobre el uso de tipado estático para mejorar la robustez del código.
* **Fundamentales:** Tipos básicos, interfaces y types.
* **Avanzado:** Genéricos, Programación Orientada a Objetos (POO) y Utility Types.
* **Integración:** Tipado de DOM y consumo de APIs.

### 2. **Vue.js 3 🟢**
Documentación enfocada en la **Composition API** y el azúcar sintáctico de `<script setup>`.
* **Reactividad:** `ref`, `reactive`, `computed` y `watch`.
* **Ciclo de Vida:** Hooks principales para el manejo de componentes.
* **Comunicación:** Props, Emits y Provide/Inject.

### 3. **Ecosistema CSS & UI 🎨**
Hojas de estilo y frameworks de utilidad para maquetación ágil.
* **Tailwind CSS 4:** Configuración de temas, directivas y diseño responsivo.
* **Bootstrap 5:** Sistema de rejilla (Grid), utilidades y componentes esenciales.
* **Flowbite:** Integración de componentes interactivos basados en Tailwind.

### 4. **Consumo de Datos 🔌**
* Manejo de peticiones asíncronas con **Axios**.
* Filtrado y transformación de datos (ej. procesamiento de la PokéAPI).

## 🛠️ Stack Tecnológico

* **Lenguajes:** TypeScript, JavaScript (ES6+), HTML5.
* **Frameworks:** Vue.js 3.
* **Estilos:** Tailwind CSS, Bootstrap, CSS3.
